### Overview
This repository contains the verified simulation notebooks for the MHH Transposition Protocol. This protocol utilizes phi-stacked magnetic resonator arrays to create a self-organizing manifold for stable state-vector transposition.

### Core Components
* `02_magnetic_core_simulation.ipynb`: Core resonance logic.
* `03_coil_design_and_resonance.ipynb`: Magnetic field modulation.
* `System Integration.ipynb`: Manifold control architecture.
* `ENGINE_DISCLOSURE.txt`: Fundamental theory and system parameters.
